# Возможности Jira Helper

[English version](./features-summary.md)

Браузерное расширение, которое расширяет возможности Jira: продвинутая визуализация, WIP-лимиты и инструменты управления потоком работ.

---

## WIP-лимиты и управление потоком

### WIP-лимиты на колонки (CONWIP)
Общий WIP-лимит на несколько колонок для визуализации Kanban-ограничений. При превышении фон колонок становится красным.

### WIP-лимиты на Swimlane
Исключение определённых swimlane (например, Expedite) из расчёта WIP колонок. Задачи в специальных swimlane имеют собственные лимиты.

### Персональные WIP-лимиты
WIP-лимиты для конкретных участников команды для балансировки нагрузки.

### WIP-лимиты по значению поля
Ограничение работы по значениям полей для Capacity Allocation. Подсчёт карточек по значениям полей или сумма числовых полей (Story Points).

### WIP-лимиты на ячейки
Лимиты на конкретные ячейки (пересечение колонки и swimlane) или группы ячеек.

## Визуализация доски

### Card Colors
Полная подсветка карточек на основе JQL-запросов вместо тонкой полоски слева. Категории карточек видны сразу.

### Swimlane Chart Bar
Визуальная диаграмма распределения задач по колонкам для каждого swimlane. Детали отображаются при наведении.

### Флажок на панели задачи
Отображение флажка задачи на странице детального просмотра (не только на доске).

### Размытие данных
Скрытие конфиденциальных данных при демонстрации доски. Включается через контекстное меню.

## Прогресс задач и связи

### Sub-tasks Progress
Прогресс-бар на карточках: статус выполнения дочерних задач, подзадач в эпиках или связанных задач. Поддержка группировки по полям (Assignee, Project и др.).

### Issue Links Display
Отображение связанных задач прямо на карточках по типу связи (Parent, Blocker и др.) с опциональной JQL-фильтрацией.

### Days in Column
Бейдж с количеством дней задачи в текущей колонке. Настраиваемые пороги предупреждения (жёлтый) и критичности (красный) для каждой колонки.

### Days to Deadline
Бейдж с оставшимися днями до дедлайна. Подсвечивает просроченные (красный) и приближающиеся дедлайны (жёлтый).

## Аналитика

### SLA-линия на Control Chart
Линия порога SLA (Service Level Agreement) на Control Chart с автоматическим расчётом процентиля.

### Линейка измерений на Control Chart
Наложение шкал измерений (Фибоначчи, линейная и др.) для анализа соответствия оценок реальному lead time.

---

## Шаблоны

### Шаблон описания задачи
Сохранение и повторное использование шаблонов описания задач. Хранятся локально в браузере для быстрого доступа при создании задач.

---

## Техническое

### Идентификация запросов
Все запросы расширения содержат специальный заголовок `browser-plugin: jira-helper/{version}` для идентификации администраторами Jira.
