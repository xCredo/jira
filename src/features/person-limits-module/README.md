# Personal WIP Limits

Контроль загрузки участников команды через индивидуальные лимиты на количество задач в работе.

Две части: настройка лимитов (Settings Page) и отображение на доске (Board Page).

## Ключевые правила

- Минимальный лимит: 1 (ноль автокорректируется)
- Дубликаты блокируются: нельзя создать два лимита с одинаковым набором фильтров
- Пустой массив фильтра = "все" (columns, swimlanes, issue types)
- Настройки хранятся в Board Property
